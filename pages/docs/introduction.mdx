## Welcome to Xcrow API documentation :-)

This API enables developers to integrate secure and reliable escrow services into their platforms.
Whether you're facilitating a straightforward transaction between two parties or implementing a gamified, condition-based contract,
the useXcrow API provides the infrastructure to safeguard user funds until predefined conditions are met.

Our system supports multiple contract types and offers flexible logic to suit use cases ranging from marketplaces to competitive wager-based engagements.
